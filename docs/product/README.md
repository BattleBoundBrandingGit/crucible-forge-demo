# Product Overview

Public **product overview package** for **The Crucible** — the AI Knowledge Operating System (AKOS) and its ecosystem (Forge, Forge Brain, Core, Aether, Barrage, Siege).

> **Public showcase only.** This package communicates vision, positioning, and high-level architecture. It does **not** include production implementation, proprietary orchestration, retrieval/ranking/embeddings, prompt compiler internals, or real client data.

## Regenerating the PDF

The PDF is generated by a standalone ReportLab script (not required to view the showcase):

```bash
python3 tools/build_pdf.py
```

Output: `docs/product/The-Crucible-Product-Overview.pdf`

Requires ReportLab and system fonts configured in the script. The committed PDF is the canonical artifact for reviewers who do not run the build locally.
