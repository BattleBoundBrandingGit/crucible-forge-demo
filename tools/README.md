# Tools

Optional utilities for maintaining showcase artifacts. **Not required** to view the demo or documentation.

| Script | Purpose |
|--------|---------|
| [`build_pdf.py`](build_pdf.py) | Regenerate `docs/product/The-Crucible-Product-Overview.pdf` via ReportLab |

```bash
python3 tools/build_pdf.py
```

Requires ReportLab and system fonts configured in the script. The committed PDF is the canonical reviewer artifact.
